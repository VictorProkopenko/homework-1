Деякі клавіатурні скорочення для macOS

Пояснення - "Cmd+C" означає копіювати, натиснули та тримаємо Cmd та натискаємо C

  • Cmd+C копіювати
  • Cmd+V вставити
  • Cmd+Y вирізати
  • Cmd+Z відмінити дію
  • Cmd+A вибрати все
  • Cmd+Q вийти з программи
  • Cmd+F пошук
  • Пробіл перегляд файла в Finder
  • Command + Shift + 3 знімок екрану
  • Command + Shift + 4 знімок виділеної області
  • Command + Shift + 4 + пробіл знімок вікна
  • Cmd+H згорнути вікно
  • Cmd+Alt+H згорнути всі вікна крім активного
  • Cmd + Tab переключення між запущенними програмами
  • Cmd+K під'єднатись до серверу
  • Cmd+Alt+Escape примусове завершення програм
  • Ctrl+Cmd+Power/Eject примусовий перезапуск комп'ютера
